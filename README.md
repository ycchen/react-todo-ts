# my-react-ts


## Start the application

`yarn install && yarn start`

#### enable console.log
```
# Add following code to tslint.json, then restart the app
  "rules": {
    "no-console": false //Disable for typescript
  },
  "jsRules": {
    "no-console": false //Disable for javascript
  }
```
